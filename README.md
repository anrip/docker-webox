# Feature

lnmp server based on docker of ubuntu, debian or alpine

contains the following modules: nginx, mysql, redis, php5/php7

# Quikc Start

```shell
docker pull vmlu/webox
```

## start and visit `http://localhost`

```shell
docker run -d -p 80:80 vmlu/webox
```
